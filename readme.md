## MapboxTab

本项目是受[MapTab](https://github.com/bsudekum/MapTab)的启发而创建的   

修复了原项目的时间不更新的Bug,将地图的随机场景切换到了中国 

使用了MapBox的矢量切片底图

![](readmeimgs/1.gif)
![](readmeimgs/2.gif)

## 如何使用
将本项目clone或下载下来,然后如下图所示  
![](readmeimgs/3.png)

在本项目release稳定版本后将提交到Chrome商店 ,届时直接安装即可
